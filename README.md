# auto-rename-tag #

Automatically rename paired HTML/XML tag.


## Usage ##
Enable for all buffers.
```el
(global-auto-rename-tag-mode t)
```
Or you can just enable in specific buffer you want.
```el
(auto-rename-tag-mode t)
```
